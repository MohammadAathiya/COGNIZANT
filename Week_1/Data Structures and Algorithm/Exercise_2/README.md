# E-Commerce Platform Search Function

## 📋 Overview
This project demonstrates the implementation and analysis of different search algorithms for an e-commerce platform's product search functionality.

## 🚀 Features
- Product catalog management
- Linear Search implementation (O(n))
- Binary Search implementation (O(log n))
- Search by product name
- Search by category
- Performance comparison
- Analysis and recommendations

## 🏗️ Architecture
ECommerceSearch/
├── Program.cs # Main program entry point
├── Product.cs # Product entity model
├── SearchService.cs # Search algorithms implementation
├── ECommerceSearch.csproj # Project configuration
└── README.md # Documentation
